# JS_ALGORITHM_STUDY
코딩테스트 준비 with JS

### 📅 스터디 일정
매주 화요일 저녁 오후 9시, Google Meet 로 진행 

### 📚 스터디 방식

1. 매주 각자가 가져온 문제를 취합해 그 중 5개를 정하고 푼다 <br/>
2. 화요일 미팅에서 각자 한 문제씩 맡아 로직 공유 및 발표를 한다 
3. 이후 어려운 부분 및 논의사항에 대해 서로 구체적으로 나눠본다
